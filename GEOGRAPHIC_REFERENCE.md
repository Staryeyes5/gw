## 添加新标记点的代码说明

### 数据结构

在 `about.html` 中找到 `deliveryCenters` 数组，每个对象包含：

```javascript
{
    name: '城市名称',      // 显示名称
    desc: '详细地址',      // 描述信息
    lat: 纬度,            // 纬度 (-90 to 90)
    lng: 经度,            // 经度 (-180 to 180)
    region: '区域名称'    // 区域分类
}
```

### 添加新城市的步骤

1. 在 `deliveryCenters` 数组中添加新的城市对象
2. 在页面底部的交付中心卡片区域添加对应的 HTML 卡片
3. 确保经纬度使用正确的格式

### 经纬度查找方法

1. 使用 Google Maps: 右键点击位置 -> 选择经纬度
2. 使用在线工具: https://www.latlong.net/
3. 参考上方的城市经纬度表

## 技术实现说明

### 纹理加载

使用 `THREE.TextureLoader` 异步加载纹理，支持失败回退。

### 坐标转换

使用 `latLngToVector3` 函数将经纬度转换为3D坐标：

```javascript
function latLngToVector3(lat, lng, radius) {
    const phi = (90 - lat) * (Math.PI / 180);
    const theta = (lng + 180) * (Math.PI / 180);
    
    const x = radius * Math.sin(phi) * Math.cos(theta);
    const y = radius * Math.cos(phi);
    const z = -radius * Math.sin(phi) * Math.sin(theta);
    
    return new THREE.Vector3(x, y, z);
}
```

### 背面隐藏逻辑

当标记点旋转到球背面时，通过以下逻辑隐藏：

```javascript
const rotatedPos = basePos.clone().applyQuaternion(globe.quaternion);
if (rotatedPos.z < 0) {
    label.visible = false;  // 球背面隐藏
} else {
    label.visible = true;   // 球正面显示
}
```

## 故障排除

### 纹理加载失败
- 检查网络连接
- 纹理会自动回退到纯色材质
- 可以替换为本地纹理文件

### 标记点位置不准确
- 确认经纬度格式正确
- 纬度范围: -90 到 90
- 经度范围: -180 到 180

### 云层不同步旋转
- 检查 `clouds.rotation` 是否与 `globe.rotation` 同步更新
